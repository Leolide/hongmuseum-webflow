Navbar Loader Example

This page demonstrates how to use the reusable navbar component.

Usage Instructions / 使用说明

Method 1: Auto-detect (Recommended)

<script src="js/navbar-loader.js?v=20260202d"></script>
<script>
  // Auto-detects language from filename and sets up language switch
  autoDetectAndInitNavbar();
</script>

Method 2: Manual Configuration

<script src="js/navbar-loader.js?v=20260202d"></script>
<script>
  NavbarLoader.init({
    lang: 'en',  // 'en' or 'cn'
    languageSwitchUrl: 'index.html'  // URL for language toggle
  });
</script>

Method 3: With Container

<div id="navbar-container"></div>
<script src="js/navbar-loader.js?v=20260202d"></script>
<script>
  NavbarLoader.init({ lang: 'cn' });
</script>

Dynamically Update Content

// Add a new exhibition
NavbarLoader.updateExhibitions('en', [
  { href: 'new-exhibition-en.html', label: 'New Exhibition', isOngoing: true },
  { href: 'digestive-ethics-en.html', label: 'Digestive Ethics' },
  // ... more exhibitions
]);

// Then re-initialize
NavbarLoader.init({ lang: 'en' });
